# SkillUp

A full-stack application with a frontend and backend.

## Project Structure
- `frontend/`: The React/Vite/Next.js frontend.
- `backend/`: The Node.js/Express backend.

## Getting Started
### Prerequisites
- Node.js
- npm

### Installation
1. Clone the repository.
2. Install dependencies in both folders.
   ```bash
   cd frontend && npm install
   cd ../backend && npm install
   ```
3. Run the development servers.
   ```bash
   # In frontend folder
   npm run dev
   # In backend folder
   npm run dev
   ```
